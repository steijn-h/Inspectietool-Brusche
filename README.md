# Inspectietool Brusche

PWA-versie van de inspectietool voor meterkasten, Scope 8, Scope 10 en Scope 12 rapportages.

## Testen lokaal
Open de map in VS Code en klik op **Go Live** met de Live Server-extensie. Gebruik bij voorkeur `http://localhost:5500` of `http://127.0.0.1:5500` in plaats van het bestand direct via `file://` te openen.

## Testen op iPad
Publiceer de repository via GitHub Pages. Open de HTTPS-link in Safari op de iPad en kies **Delen → Zet op beginscherm**.

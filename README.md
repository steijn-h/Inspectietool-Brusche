# Inspectietool Brusche

PWA-versie van de inspectietool.

## Testen lokaal
Open de map in VS Code en klik op **Go Live** met de Live Server-extensie.

## Testen op iPad
Publiceer de repository via GitHub Pages. Open de HTTPS-link in Safari op de iPad en kies **Delen → Zet op beginscherm**.

## PWA-bestanden
- `manifest.webmanifest`
- `service-worker.js`
- `assets/icons/icon-192.png`
- `assets/icons/icon-512.png`
